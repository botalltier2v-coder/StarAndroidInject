# StarAndroidInject

A powerful Android Native Injector and Mod Menu template using C++ and Kotlin.

### 🛠 Tech Stack
- **Languages:** Kotlin (UI/Logic), C++ (Native Modding), C (System), Assembly (ARM64 Patches).
- **Interface:** JNI (Java Native Interface) & Jetpack Compose.
- **Libraries:** KittyMemory (Memory Patching), DobbyHook (Function Hooking).
- **Build System:** CMake & Gradle.

### 🚀 Features
- Memory Offset Patching (Hex).
- Function Hooking (Dobby).
- Multi-threaded Socket Control.
- il2cpp Game Support.
