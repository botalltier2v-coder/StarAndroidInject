# StarAndroidInject
A powerful Android Native Injector and Mod Menu template using C++ and Kotlin.

### 🛠 Tech Stack
- **Languages:** Kotlin, C++, C, ARM64 Assembly
- **Interface:** JNI & Jetpack Compose
- **Libraries:** KittyMemory & DobbyHook
